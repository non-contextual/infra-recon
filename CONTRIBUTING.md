# Contributing to Infra Recon

Thank you for your interest in contributing. This tool focuses on passive reconnaissance without generating noise.

## Workflow

1. Fork the repository and create your branch from `main`.
2. Ensure new modules adhere to the existing `_module.py` interfaces.
3. Test your module against a safe, authorized target.
4. Submit a pull request.

## Pull Request Process

- Ensure your PR description clearly explains the new OSINT source or bug fix.
- Keep commits atomic and descriptive (e.g., `feat: add Sublist3r API support`).
- The PR will be reviewed based on passive-only constraints and utility.
